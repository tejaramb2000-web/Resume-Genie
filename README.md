# ResumeTailorApp

An AI-powered ASP.NET Core application that helps users optimize resumes based on job descriptions using intelligent matching and rewriting.

---

## Overview

ResumeTailorApp is designed to improve job application success by aligning resumes with specific job requirements. The system analyzes job descriptions, compares them with uploaded resumes, and generates optimized versions using AI services.

---

## Key Features

- Upload and manage resumes
- Add and track job descriptions
- Match resumes with job requirements
- AI-powered resume rewriting
- Resume version history tracking
- User authentication (Login / Register)
- Export resumes (PDF / DOCX)

---

## Tech Stack

- **Backend:** ASP.NET Core (C#)
- **Database:** PostgreSQL (Entity Framework Core)
- **Frontend:** Razor Views, Bootstrap
- **AI Integration:** Groq, Ollama
- **Architecture:** MVC Pattern

---

## Architecture Overview

- MVC Pattern (Controllers, Models, Views)
- Service layer for AI processing and document export
- Entity Framework Core for database operations
